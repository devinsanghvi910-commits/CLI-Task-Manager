# CLI To-Do List (Python)
## Project Url: 
https://github.com/devinsanghvi910-commits/CLI-Task-Manager
## Description
This is a simple command-line to-do list application written in Python.
It allows users to add tasks, remove tasks, view all current tasks, and quit the program using a text-based menu.
## Features
Add new tasks
Remove existing tasks by name
Display all active tasks
Input validation for invalid menu choices
## How It Works
When the program starts, it displays a menu with available options.
The program continues running in a loop until the user chooses to quit.
All tasks are stored in memory during runtime.
## How to Run
Make sure Python 3 is installed.
Run the program from the terminal:
python todo.py
Follow the on-screen prompts.
